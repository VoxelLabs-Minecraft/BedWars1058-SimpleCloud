BedWars1058 Server-ID Auto Setup (SimpleCloud Extension)
This plugin is a SimpleCloud extension designed to work seamlessly with BedWars1058. It automatically sets the Server ID for BedWars1058 instances on startup, removing the need for manual configuration and reducing setup errors in dynamic cloud environments.

Perfect for networks using SimpleCloud with automatically deployed BedWars servers, this plugin ensures that every server receives a correct and unique Server ID based on its cloud service configuration.

✨ Features
✅ Automatically sets the BedWars1058 Server ID on server startup

☁️ Fully integrated with SimpleCloud

⚡ Ideal for dynamic and scalable BedWars server setups

No manual configuration required

Lightweight and easy to use

Use Case
If you are running BedWars servers in a cloud-based environment where servers are created and stopped dynamically, this plugin ensures that BedWars1058 always uses the correct Server ID without any manual intervention.
